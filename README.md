# Power BI Dashboards

Repozytorium zawiera zestaw dashboardów utworzonych w ramach nauki i doskonalenia umiejętności pracy z Power BI.  
Każdy projekt prezentuje inne zagadnienie analityczne oraz wykorzystanie różnych technik wizualizacji, modelowania danych i tworzenia miar DAX.

# Dostępne projekty

Poniżej przygotowane w ramach nauki dashboardy, każdy z własnym plikiem README.md.

## Data Job Dashboard (v1 nauka podstawowych elementów programu)
![Page1](/Data_Jobs_v1/Assets/Page1.png)

### Zakres umiejętności

 - Czyszczenie danych w Power Query
 - Tworzenie i konfiguracja KPI
- Projektowanie różnych typów wykresów
- Budowa tabel i macierzy
- Wykorzystanie slicerów i filtrów
- Tworzenie miar (measures) w DAX
- Dodawanie przycisków i zakładek (bookmarks)
- Projektowanie intuicyjnego interfejsu użytkownika w Power BI
- Drill-Through
- Visual Studio Code
- Git & GitHub

[Zobacz szczegóły projektu (README.md)](/Data_Jobs_v1/README.md)
