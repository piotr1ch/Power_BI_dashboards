# Power BI Dashboards

Repozytorium zawiera zestaw dashboardów utworzonych w ramach nauki i doskonalenia umiejętności pracy z Power BI.  
Każdy projekt prezentuje inne zagadnienie analityczne oraz wykorzystanie różnych technik wizualizacji, modelowania danych i tworzenia miar DAX.
