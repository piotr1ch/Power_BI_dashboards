# Power BI Dashboard Data Jobs v2

## Opis projektu

![Page1](/Assets/Page1.png)


Projekt przedstawia końcowy dashboard stworzony w Power BI na podstawie danych z 2024 roku dotyczących rynku pracy w obszarze analizy danych.  
Dane pochodzą z kursu [Power BI Course](https://lukebarousse.com/powerbi) autorstwa Luke’a Barousse’a na YouTube.  
Analiza została przeprowadzona krok po kroku zgodnie z materiałem z kursu, w celu nauki i praktycznego opanowania narzędzia Power BI.

## Opis dashboardu

![Page1](/Data_Jobs_v2/Assets/Project_2_Page1.png)


W oparciu o dane i rozwiązania z poprzedniego projektu uproszczono sposób prezentacji informacji, dając użytkownikowi możliwość spojrzenia na rynek pracy z perspektywy wymaganych umiejętności oraz poziomu wynagrodzeń.  
Wszystkie kluczowe dane zostały zebrane na jednej, przejrzystej stronie, co ułatwia szybkie wnioskowanie bez konieczności przeskakiwania między zakładkami.

## Zakres umiejętności

- Estetyczne i funkcjonalne projektowanie dashboardów  
- Czyszczenie i przygotowanie danych w Power Query  
- Modelowanie danych (star schema)  
- Podstawy języka DAX  
- Praca z Visual Studio Code  
- Wykorzystanie Git i GitHub do wersjonowania projektów  

## Podsumowanie

Druga wersja projektu pokazuje, że nawet pojedyncza strona dashboardu może skutecznie przekazać wiele informacji -  
w przejrzysty i intuicyjny sposób, bez konieczności przeglądania wielu różnych wykresów i zakładek.
