# Power BI Dashboard Data Jobs

## Opis projektu

![Page1](/Data_Jobs_v1/Assets/Page1.png)


Projekt to dashboard stworzony w Power BI na podstawie danych z 2024 roku dotyczących rynku pracy w obszarze analizy danych. Dane pochodzą z kursu [Power BI Course](https://lukebarousse.com/powerbi) autorstwa Luke’a Barousse’a na YouTube. Analiza została przeprowadzona krok po kroku zgodnie z instrukcjami z kursu, w celu nauki i praktycznego opanowania narzędzia Power BI.

## Opis dashboardu

### Strona 1:
![Page1](/Data_Jobs_v1/Assets/Page1.png)

Dashboard przedstawia ogólny obraz rynku pracy w 2024 roku, obejmując liczbę ofert pracy oraz mediany zarobków rocznych i godzinowych (KPI). Pokazuje również różnice między najczęściej poszukiwanymi stanowiskami a tymi, które oferują wyższe wynagrodzenie, umożliwiając lepsze zrozumienie aktualnych trendów w branży analizy danych.

### Strona 2:
![Page2](/Data_Jobs_v1/Assets/Page2.png)

Z poziomu dashboardu dostępna jest funkcja drill-through, umożliwiająca przejście do szczegółowych informacji dla wybranego stanowiska. Użytkownik może sprawdzić m.in. procent ofert z pracą zdalną, wymagających stopnia naukowego lub oferujących ubezpieczenie zdrowotne, a także zakres wynagrodzeń, kraje najczęściej publikujące oferty oraz platformy, na których są one dostępne.

## Zakres umiejętności

 - Czyszczenie danych w Power Query
 - Tworzenie i konfiguracja KPI
- Projektowanie różnych typów wykresów
- Budowa tabel i macierzy
- Wykorzystanie slicerów i filtrów
- Tworzenie miar (measures) w DAX
- Dodawanie przycisków i zakładek (bookmarks)
- Projektowanie intuicyjnego interfejsu użytkownika w Power BI
- Drill-Through
- Visual Studio Code
- Git & GitHub

## Podsumowanie

Projekt pozwolił mi lepiej zrozumieć trendy na rynku pracy w branży analizy danych w 2024 roku — które stanowiska są najbardziej poszukiwane, a które oferują najwyższe wynagrodzenie.
Podczas realizacji projektu nauczyłem się praktycznie wykorzystywać funkcje Power BI.
